An R package of algorithms for estimation of the graphical lasso regularization parameter.

Accompanying paper: Pedro Cisneros-Velarde, Alexander Petersen and Sang-Yun Oh (2020). Distributionally Robust Formulation and Model Selection for the Graphical Lasso. AISTAT 2020.

Installation

To install from CRAN

install.packages("robsel")

Example

This is a basic example which shows basic usage of the robsel package to estimate of the regularization parameter for Graphical Lasso.

library(robsel)
## basic example code
x <- matrix(rnorm(100*5),ncol=5)

Estimate lambda for glasso

lambda <- robsel(x, alpha=0.9, B=200)
lambda
#> [1] 0.1561845

Use glasso directly with robsel estimate

glasso.model <- robsel.glasso(x, alpha=0.9)
